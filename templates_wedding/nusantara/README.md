# Nusantara Batik Wedding Template

Template undangan pernikahan dengan tema Tradisional Nusantara (Batik & Jawa).

## Cara Menggunakan
1. Buka file `index.html` dengan browser.
2. Template ini menggunakan desain elegan dengan warna emas dan coklat batik.

## Cara Kustomisasi
Untuk mengubah data, buka file `index.html` dengan text editor:

- **Nama Pasangan:** Cari teks "Rama" dan "Shinta".
- **Foto:** Cari link gambar (`https://images.unsplash.com/...`) di bagian `#mempelai`.
- **Tanggal & Lokasi:** Cari bagian "Rangkaian Acara" (`#Acara`).
- **Musik:** Ganti URL di tag `<audio>` dengan link Gamelan atau lagu tradisional favorit.

## Fitur Utama
- **Nuansa Emas & Batik:** Menggunakan palet warna premium (Gold & Batik Brown).
- **Opening Animasi:** Efek fade-in yang halus saat dibuka.
- **Font Klasik:** Menggunakan 'Cormorant Garamond' dan 'Great Vibes' untuk kesan ningrat.
- **Responsif:** Tampilan menyesuaikan HP dan Laptop.
