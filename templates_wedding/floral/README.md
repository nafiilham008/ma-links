# Floral Romantic Wedding Template

Template undangan pernikahan dengan tema bunga-bunga yang cantik, lembut, dan romantis.

## Cara Menggunakan
1. Buka file `index.html` dengan browser.
2. Template ini mengusung warna pastel (Pink & Sage Green) yang kekinian.

## Cara Kustomisasi
Untuk mengubah data, edit file `index.html`:

- **Nama Pasangan:** Cari teks "Adam" dan "Hawa".
- **Foto:** Cari link gambar (`https://images.unsplash.com/...`) di bagian `#mempelai`. Ganti dengan foto prewedding Anda.
- **Tanggal & Lokasi:** Cari bagian "Rangkaian Acara".
- **Musik:** Ganti URL di tag `<audio>` untuk mengubah lagu.

## Fitur Utama
- **Desain Pastel:** Warna Rose Pink & Sage Green yang lembut.
- **Animasi Bunga:** Hiasan bunga yang bergerak halus (floating effect).
- **Typography Mewah:** Kombinasi font *Script* (latin) dan *Serif* yang elegan.
- **Scroll Reveal:** Elemen muncul perlahan saat di-scroll.

Cocok untuk tema pernikahan outdoor, garden party, atau modern romantic.
