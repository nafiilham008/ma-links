# Spotify Wedding Template

Template undangan pernikahan dengan tema musik / Spotify.

## Fitur Unik
- **Layout Playlist:** Susunan acara ditampilkan seperti daftar lagu (*tracklist*).
- **Dark Mode:** Tema gelap dengan aksen hijau Spotify yang elegan.
- **Player Bar:** Kontrol musik di bagian bawah yang melayang (sticky).
- **Album Cover:** Foto mempelai dijadikan cover album besar di header.

## Cara Menggunakan
1. Buka file `index.html` di browser.
2. Klik tombol "Play" hijau besar untuk memutar musik.
3. Klik pada baris lagu (Track) untuk melihat detail acara.

## Cara Kustomisasi
- **Cover Album:** Cari `img` di bagian header lalu ganti `src`-nya dengan foto mempelai.
- **Lagu / Audio:** Ganti file audio di tag `<audio id="bg-audio">`.
- **Daftar Acara:** Edit bagian `class="track-row"`. 
  - Kolom pertama: Nomor urut.
  - Kolom kedua: Nama Acara (Judul Lagu) & Detail (Artis/Keterangan).
  - Kolom kanan: Durasi / Waktu.

## Font
Template ini menggunakan font 'Inter' yang mirip dengan font asli Spotify (Circular).
