# KAI Train Ticket Wedding Template

Template undangan pernikahan dengan tema Tiket Kereta Api (KAI Boarding Pass).

## Konsep Desain
- **KAI Branding:** Menggunakan warna Biru (#005C9F) dan Oranye (#F37021) khas KAI.
- **Istilah Perkeretaapian:**
  - *Nama Kereta* = "ARGO ASMARA" (Bisa diubah jadi nama kalian)
  - *Stasiun Awal (SGL)* = Stasiun Jomblo / Single
  - *Stasiun Tujuan (MRD)* = Stasiun Halal / Married
  - *Check-in* = RSVP
- **Fitur Khas:** Barcode, QR Code, dan Running Text (Marquee) mirip display stasiun.

## Cara Menggunakan
1. Buka file `index.html` di browser.
2. Klik tombol "KONFIRMASI KEHADIRAN" untuk membuka form Check-in Mandiri.

## Cara Kustomisasi
- **Nama Kereta:** Cari teks "ARGO ASMARA" dan ubah sesuai selera (misal: "KA JODOH", "TURANGGA LOVE").
- **Kode Booking:** Ubah teks "CINTA26" menjadi inisial nama dan tanggal.
- **Rute:** Ubah keterangan "Stasiun Jomblo" dan "Stasiun Halal".
- **Barcode:** Menggunakan font 'Libre Barcode 128', teks barcode ada di footer.

Sangat cocok untuk Railfans atau pasangan yang punya kenangan di kereta api!
