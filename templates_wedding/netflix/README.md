# Netflix Wedding Template

Template undangan pernikahan dengan tema Netflix.

## Cara Menggunakan
1. Buka file `index.html` dengan browser (Chrome, Firefox, Edge, dll).
2. Klik profil "Tamu Undangan" untuk masuk ke dalam undangan.

## Cara Kustomisasi
Untuk mengubah data, buka file `index.html` dengan text editor (Notepad, VS Code, dll) dan ubah bagian berikut:

- **Nama Pasangan:** Cari teks "Romeo" dan "Juliet".
- **Foto:** Cari link gambar (`https://images.unsplash.com/...`) dan ganti dengan link foto Anda. 
  - Tips: Anda bisa hosting foto di imgur, google photos, atau folder lokal.
- **Tanggal & Lokasi:** Cari bagian "Episodes (Rangkaian Acara)".
- **Musik:** Ganti URL di tag `<audio>` dengan link lagu MP3 pilihan Anda.

## Struktur
- **Splash Screen:** Halaman awal "Who's watching".
- **Hero:** Tampilan utama ala film.
- **Episodes:** Rangkaian acara (Akad, Resepsi) ditampilkan seperti episode serial.
- **Gallery:** Bagian "Trailers & More".
- **RSVP:** Form konfirmasi kehadiran.

Dibuat dengan ❤️ menggunakan HTML & Tailwind CSS.
