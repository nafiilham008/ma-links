# Travel Boarding Pass Wedding Template

Template undangan pernikahan dengan tema Tiket Pesawat / Boarding Pass.

## Konsep Desain
- **Boarding Pass Style:** Menggunakan layout kartu yang menyerupai tiket pesawat fisik.
- **Flight Details:** Istilah pernikahan diubah menjadi istilah penerbangan:
  - *Passenger* = Tamu Undangan
  - *From SGL to MRD* = Single to Married Life (Perjalanan Cinta)
  - *Boarding Time* = Waktu Acara
  - *Gate* = Lokasi / Gedung
- **Barcode & Stamps:** Hiasan barcode dan stempel "Approved" menambah kesan realistik.

## Cara Menggunakan
1. Buka file `index.html` di browser.
2. Klik tombol "ONLINE CHECK-IN" untuk membuka form RSVP.

## Cara Kustomisasi
- **Logo Penerbangan:** Edit bagian "Wedding Airlines" di header.
- **Rute:** Ubah kode bandara "SGL" (Single) dan "MRD" (Married) sesuai keinginan.
- **Foto Paspor:** Ganti `img` di bagian "The Captains" dengan foto mempelai.
- **Barcode Font:** Menggunakan Google Font 'Libre Barcode 39 Text', teks barcode bisa diedit di bagian footer.

Tema ini sangat cocok untuk pasangan yang hobi traveling atau LDR (Long Distance Relationship).
